# writing-japanese-work-replies

## Purpose
Draft concise, polite, natural Japanese business messages that fit the requested tone.

## When to use
- thank-you replies
- schedule coordination
- completion acknowledgements
- status updates
- tone softening
- shortening a draft while keeping politeness

## Tone defaults
- polite
- natural
- not overly stiff
- no unnecessary repetition
- no mixed languages unless requested

## Workflow
1. Identify the message type:
   - reply
   - acknowledgement
   - scheduling
   - apology
   - request
2. Identify the relationship:
   - professor / university
   - company / recruiter
   - vendor / contractor
   - field operator / stakeholder
3. Keep the message short unless expansion is requested.
4. Preserve the user’s intended meaning.
5. Remove awkward literal phrasing.
6. Ensure the closing line matches the message purpose.

## Output options
- formal version
- slightly softer version
- shorter version
- very concise version

## Verification
- Japanese is natural
- tone matches recipient
- meaning is preserved
- unnecessary verbosity is removed
- no character corruption or language mixing
